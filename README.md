# 📑 Desafios de Algoritmos: Pseudocódigo e Fluxograma

Este repositório contém o conjunto de atividades práticas desenvolvidas para o estudo de **Lógica de Programação**. O projeto foca na transposição de tarefas cotidianas e cálculos matemáticos para o formato de algoritmos estruturados, utilizando técnicas de representação textual (pseudocódigo) e visual (fluxograma).

---

## Objetivo do Projeto

O objetivo principal foi exercitar a **decomposição de problemas**, transformando processos de cálculos e condições em sequências lógicas, finitas e ordenadas, garantindo a compreensão clara de cada etapa necessária para atingir um resultado esperado.

## 🛠️ Metodologia e Ferramentas

Para a elaboração desta atividade, foram utilizadas as seguintes ferramentas:

* **Pseudocódigo:** Escrita lógica em linguagem estruturada (Portugol) para descrever os passos de cada tarefa, executada no software **VisuAlg**.
* **Fluxograma:** Representação gráfica da sequência operacional, desenvolvida utilizando a ferramenta **Draw.io**, facilitando a visualização do fluxo de dados e tomadas de decisão.

## Atividades Desenvolvidas

Abaixo estão os módulos contemplados neste repositório:

* **Cálculo de Área:** Trapézio e Terrenos.
* **Saúde e Bem-estar:** Cálculo de Índice de Massa Corporal (IMC).
* **Matemática e Conversão:** Soma de números, cálculo de médias e conversão de graus.
